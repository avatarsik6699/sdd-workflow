# Support

## Where to ask for help

- Usage questions and troubleshooting: open a GitHub Discussion in this repo.
- Bug reports: open a GitHub Issue using the bug template.
- Security concerns: follow [SECURITY.md](SECURITY.md).

## What to include

- What you ran (`/workflow-init`, `/spec-init`, etc.)
- Relevant file paths and snippets
- Expected vs actual behavior
- Steps to reproduce
