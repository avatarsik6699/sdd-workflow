# Example: minimal integrated project

This folder documents the expected shape of a project after running `/workflow-init`.

Expected top-level additions:

- `AGENTS.md`
- `CLAUDE.md`
- `.claude/skills/` (5 skills)
- `plugins/sdd-workflow/`
- `docs/playbooks/`
- `docs/SPEC.md`, `docs/STATE.md`, `docs/CONTEXT.md`, `docs/STACK.md`,
  `docs/CHANGELOG.md`, `docs/PHASE_TEMPLATE.md`, `docs/KNOWN_GOTCHAS.md`, `docs/DECISIONS.md`

Use this as a structural reference for docs and demos.
