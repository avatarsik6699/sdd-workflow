# Security Policy

## Supported Versions

This repository maintains one active version line on `main`.

| Version | Supported |
| --- | --- |
| `main` | :white_check_mark: |
| Older tags | :x: |

## Reporting a Vulnerability

Please do not open public issues for potential security problems.

1. Send a report to `security@sdd-workflow.example`.
2. Include impact, affected files, and reproduction steps.
3. If possible, include a proposed fix.

We aim to acknowledge reports within 3 business days and provide a remediation
plan within 10 business days.

After a fix is available, we will publish a changelog entry and, when needed,
a security advisory.
