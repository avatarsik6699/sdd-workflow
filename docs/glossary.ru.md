# Глоссарий SDD

- **SPEC**: продуктовый контракт и базовая область работ.
- **PHASE**: ограниченный срез реализации, производный от SPEC.
- **Gate**: исполняемые проверки из `docs/STACK.md`.
- **Context update**: синхронизация рабочей документации после реализации.
- **Playbook**: канонический процедурный документ для одного навыка.
