# Совместимость

## Поддерживаемые версии агента/окружения

| Компонент | Поддержка |
| --- | --- |
| Claude Code | Актуальный стабильный канал с поддержкой slash-команд навыков |
| Codex plugin runtime | Актуальный стабильный канал с поддержкой skills/commands |
| Bash | POSIX-совместимая оболочка с доступным `bash` |

Целевая совместимость поддерживается на `main`. Старые теги могут не содержать новых обёрток или playbooks.
